# 🚀 快速啟動指南

## 方法一：直接在瀏覽器中打開（不推薦）

如果您直接雙擊 `index.html`，會因為瀏覽器的 CORS 安全策略而無法加載 ES6 模塊。

**建議使用下面的方法二或方法三。**

## 方法二：使用 Python（推薦）

### 前提條件

- 已安裝 Python 3.x

### 步驟

1. **打開終端或命令提示符**  
   Windows: 使用 PowerShell 或 CMD

2. **進入項目目錄**

   ```bash
   cd d:\yueting.zhou\Project\background-remover
   ```

3. **啟動本地服務器**

   ```bash
   python -m http.server 8000
   ```

4. **打開瀏覽器訪問**  
   在瀏覽器中打開: `http://localhost:8000`

5. **停止服務器**  
   按 `Ctrl + C`

## 方法三：使用 Node.js

### 前提條件

- 已安裝 Node.js

### 步驟

1. **打開終端**

   ```bash
   cd d:\yueting.zhou\Project\background-remover
   ```

2. **使用 npx 啟動服務器**

   ```bash
   npx http-server -p 8000
   ```

   或使用 npm script:

   ```bash
   npm start
   ```

3. **打開瀏覽器訪問**  
   `http://localhost:8000`

## 方法四：使用 VS Code Live Server

### 前提條件

- 已安裝 Visual Studio Code
- 已安裝 Live Server 擴展

### 步驟

1. **在 VS Code 中打開項目文件夾**

   ```
   File → Open Folder → 選擇 background-remover 文件夾
   ```

2. **右鍵點擊 index.html**
   選擇 "Open with Live Server"

3. **瀏覽器將自動打開應用**

## 使用說明

### 1. 上傳圖片

- 點擊「上傳圖片」按鈕
- 選擇您的圖片文件（支持 JPG, PNG, GIF 等）

### 2. 調整參數

- **容差 (0-100)**：數值越大，移除的背景區域越多
- **邊緣平滑 (0-10)**：數值越大，邊緣越平滑
- **邊緣羽化 (0-5)**：數值越大，邊緣過渡越柔和

### 3. 移除背景

- 在左側「原圖」畫布上點擊要移除的背景區域
- 右側「處理結果」將顯示透明背景的圖片

### 4. 顏色替換（可選）

- 勾選「替換顏色」複選框
- 選擇來源顏色（要替換的顏色）
- 選擇目標顏色（替換後的顏色）
- 調整顏色容差

### 5. 下載結果

- 點擊「下載結果」按鈕
- 圖片將以 PNG 格式（支持透明）保存到您的下載文件夾

## 技巧和建議

### 💡 獲得最佳效果

1. **選擇合適的圖片**
   - ✅ 背景顏色單一或接近單一
   - ✅ 前景與背景對比明顯
   - ❌ 背景過於複雜

2. **調整容差值**
   - 純色背景：20-30
   - 輕微漸變：30-50
   - 複雜背景：50-70

3. **使用邊緣處理**
   - 邊緣平滑：去除鋸齒
   - 邊緣羽化：柔化過渡

4. **多次嘗試**
   - 如果效果不理想，點擊「重置」
   - 調整參數後再次點擊背景

## 常見問題

### Q: 為什麼直接打開 HTML 文件不工作？

A: 由於瀏覽器的 CORS 安全策略，ES6 模塊需要通過 HTTP(S) 協議加載。必須使用本地服務器。

### Q: 如何移除多個背景區域？

A: 逐個點擊不同的背景區域。每次點擊都會處理一個連續的顏色區域。

### Q: 為什麼有些背景沒有被移除？

A: 可能原因：

- 容差值太小 → 增加容差值
- 背景顏色差異大 → 分多次點擊不同區域
- 前景與背景顏色相似 → 這種情況較難處理

### Q: 如何獲得更平滑的邊緣？

A:

1. 增加「邊緣平滑」值（3-5）
2. 增加「邊緣羽化」值（1-3）
3. 使用更高質量的原圖

### Q: 下載的圖片背景不透明？

A:

- 確保下載的是 PNG 格式（自動）
- 某些應用可能不支持透明度
- 嘗試在支持透明度的軟體中打開（如 Photoshop、GIMP）
